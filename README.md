# BillFlow AI

BillFlow AI is a modern invoice management platform that helps freelancers and small businesses create, track, and manage invoices efficiently.

## Current Features

* User authentication with Supabase
* Create invoices
* Track invoice status
* Dashboard overview
* Responsive UI
* Vercel deployment

## Planned Features

* Stripe payment integration
* AI invoice extraction
* WhatsApp invoice parsing
* Automated payment reminders
* PDF invoice generation

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Supabase
* PostgreSQL
* Vercel

## Getting Started

```bash  
git clone https://github.com/Gani-767/billflow-ai.git
cd billflow-ai
npm install
npm run dev
```

## Deployment

Deployed on Vercel.
